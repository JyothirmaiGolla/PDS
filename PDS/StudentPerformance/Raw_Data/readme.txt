Gender: This category denotes the student's sex, with options "male" and "female."

Race/Ethnicity: This attribute specifies the racial or ethnic group to which the student belongs.

Parental Education Level: This data point represents the highest educational attainment of the student's parents.

Lunch Type: This data field indicates the kind of meal plan the student receives, with choices of "standard" or "free/reduced."

Test Prep Completion: This field indicates whether the student has completed a test preparation course, with options "completed" and "none."

Math Achievement: This field contains the student's score in the mathematics subject.

Reading Achievement: This field contains the student's score in the reading subject.

Writing Achievement: This field contains the student's score in the writing subject.
